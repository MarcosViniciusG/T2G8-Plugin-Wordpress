Como Instalar e Ativar o Plugin de Georreferenciamento LGBTQ+ no WordPress

  1. Passo 1: Faça o Download do Plugin
    2. 1.1 - Primeiramente, baixe o arquivo do plugin LGBTQ+ Connect no formato .zip a partir do nosso site oficial ou da página do plugin no repositório do WordPress.
  2. Passo 2: Acesse o Painel Administrativo do WordPress
    3. 2.1 Entre no painel administrativo do seu site WordPress. Normalmente, isso pode ser feito acessando www.seusite.com/wp-admin.
  3. Passo 3: Instale o Plugin
    4. 3.1 - No menu lateral esquerdo, clique em "Plugins"
    3.2 - Ao entrar na área de plugins, clique em "Adicionar Plugin"
    3.3 - Clique no botão "Enviar Plugin" no topo da página
    3.4 - Clique no botão "Escolher arquivo"
    3.5 - Selecione o arquivo .zip do plugin baixado anteriormente
    3.6 - Clique em Instalar agora
    3.7 - Após a instalação ser concluída, clique no botão "Ativar Plugin".
    3.8 - Com isso o plugin LGBTQ+ Connect finalmente está ativado em seu site!
